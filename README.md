Laravel Authentication Application

This project is a full-stack Laravel application implementing authentication, user roles, and content management. It provides full CRUD operations for posts and comments, middleware-protected routes, and a modular architecture following Laravel best practices.

Features
- User authentication (Laravel Breeze)
- CRUD for Posts and Comments
- Admin/user roles
- Tailwind CSS UI components
- API endpoints

Tech Stack

- Laravel 12
- PHP 8.4
- Tailwind CSS
- MySQL
- Blade/Breeze
- Github

Installation & Setup

git clone https://github.com/Facenam3/laravel-auth-app.git

cd project-folder

configure your .env.example/.env

composer install

php artisan key:generate

php artisan migrate

npm install

npm run dev

php artisan serve
